# deep-research — 深度调研

本地可跑的调研闭环：对比、选型、综述、竞品/技术调研，报告要带引用。

## 何时用

| 要做 | 用 |
|------|----|
| 调研 / 对比 / 选型 / 综述 | **本技能** |
| 创意 / 未定型方案 | `brainstorming` |
| 写实施计划 | `spec-driven-development` |
| 系统审查 | `system-review` |
| 编码落地 | `loop` / `go` |

## 怎么开

```
调研 FastAPI 依赖注入
对比 A vs B 哪个适合我
```

也可经 `/go` 的 `research_compare` family 调度。

流程与模板见同目录 `SKILL.md`、`workflow.md`。
