# Release It! 规则集

当设计生产就绪系统、处理故障模式、规划发布策略、确保系统稳定性时使用此技能。

## 使用方式

加载此技能后，**必须**读取并严格遵循以下规则文件：

```
C:\Users\admin\.agents\skills\agent-rules-books\release-it\release-it.mini.md
```

## 规则概览

- 稳定性反模式：集成点、连锁反应、级联故障
- 稳定性模式：超时、断路器、隔舱、稳态处理
- 容量规划与容量测试
- 发布模式：蓝绿部署、金丝雀发布
- 运维透明性
- SLA 与监控

**_规则文件是此技能的核心。务必在设计前读取它。_**
