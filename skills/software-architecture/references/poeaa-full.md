# Patterns of Enterprise Application Architecture 规则集

当设计企业级应用架构、选择架构模式、处理数据源或 Web 表示层时使用此技能。

## 使用方式

加载此技能后，**必须**读取并严格遵循以下规则文件：

```
C:\Users\admin\.agents\skills\agent-rules-books\patterns-of-enterprise-application-architecture\patterns-of-enterprise-application-architecture.mini.md
```

## 规则概览

- 领域逻辑组织模式：事务脚本、表模块、领域模型
- 数据源架构模式：表数据入口、行数据入口、活动记录、数据映射器
- 对象关系（ORM）行为模式：工作单元、标识映射、延迟加载
- 对象关系结构模式：单表继承、类表继承、具体表继承
- Web 表示模式：MVC、页面控制器、前端控制器、模板视图
- 离线并发模式：乐观锁、悲观锁
- 会话状态管理
- 分布模式

**_规则文件是此技能的核心。务必在设计前读取它。_**
