# Designing Data-Intensive Applications 规则集

当设计数据密集型系统、处理分布式数据、一致性模型、复制/分区、批流处理时使用此技能。

## 使用方式

加载此技能后，**必须**读取并严格遵循以下规则文件：

```
C:\Users\admin\.agents\skills\agent-rules-books\designing-data-intensive-applications\designing-data-intensive-applications.mini.md
```

## 规则概览

- 可靠性、可扩展性、可维护性三大基石
- 数据模型与查询语言选择
- 存储引擎与索引
- 编码与演化
- 复制：主从、多主、无主
- 分区与二级索引
- 事务与隔离级别
- 分布式系统的共识与一致性
- 批处理与流处理

**_规则文件是此技能的核心。务必在设计前读取它。_**
