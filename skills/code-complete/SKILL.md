# Code Complete 规则集

当需要软件构造全过程指导——从需求分析、设计、编码到调试和测试时使用此技能。

## 使用方式

加载此技能后，**必须**读取并严格遵循以下规则文件：

```
C:\Users\admin\.agents\skills\agent-rules-books\code-complete\code-complete.mini.md
```

## 规则概览

- 为变化而设计，为理解而编码
- 管理复杂度是软件开发的核心
- 选用合适的数据结构和算法
- 高质量的子程序：内聚、命名清晰、参数合理
- 防御式编程：检查输入、断言前置条件
- 变量命名反映用途和作用域
- 控制流的结构和可读性
- 代码调优要在测量后针对热点

**_规则文件是此技能的核心。务必在编码前读取它。_**
