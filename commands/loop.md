使用 `loop` 技能执行单任务闭环：已有可执行目标 + 验收 → 编码 ↔ 门禁/验证 ↔ 自愈 → 交付。

加载 `skills/loop/SKILL.md` 获取薄执行环方法论。不做产品级需求确认或完整实施计划；未定型需求走 brainstorming / `/go`。

## 触发词

`/loop`、闭环编码、单任务

## 不适用

- 多任务并行 / 编排（用 `/go` 或 `subagent-driven-development`）
- 未定型探索 / 选型（用 `brainstorming`）
- 纯调研（用 `deep-research`）
